[![Dev](https://img.shields.io/badge/docs-dev-blue.svg)](https://ArndtLab.github.io/DemoPlots.jl/dev/)

A companion module to [HetDister.jl](https://github.com/ArndtLab/HetDister.jl)
with plotting utilities for raw data as well as inference results.

To use it together with DemoInfer, just add it in the same Julia environment:
```julia
using Pkg
Pkg.add("DemoPlots")
using DemoPlots
```
See the link above and docs for more details.
